# Student-Performance-Analysis-Dashboard
The dataset used for this project contains information related to student academic performance and influencing factors.

Project Overview

This project presents an interactive *Power BI dashboard* designed to analyze the key factors influencing student exam performance. The objective of this analysis is to explore how different academic, behavioral, and demographic variables affect students’ exam scores.

The dashboard allows users to visually explore relationships between variables such as study hours, motivation level, attendance, parental involvement, and school characteristics. Through interactive visualizations and filters, the dashboard helps identify patterns that contribute to student success.

Key Variables in the Dataset
- Exam Score – Final exam score achieved by the student
- Hours Studied – Number of hours a student spends studying
- Attendance – Student attendance percentage
- Motivation Level – Level of student motivation (High, Medium, Low)
- Parental Involvement – Level of parental engagement in education
- Parental Education Level – Highest education level of parents
- School Type – Public or Private school
- Distance from Home – Distance between student's home and school
- Gender – Student gender
- Previous Scores – Historical academic performance indicator

Dashboard Features
The Power BI dashboard includes several interactive components to support exploratory analysis:

KPI Indicators
- Average Exam Score
- Maximum Exam Score
- Average Study Hours
- Average Attendance

Visualizations
- Scatter Plot: Study Hours vs Exam Score (with Motivation Level segmentation)
- Score Distribution Chart: Distribution of exam scores with previous score comparison
- Bar Charts:  
  - Average Exam Score by Hours Studied  
  - Average Exam Score by Parental Involvement
- Donut Charts: 
  - Distribution of Parental Involvement  
  - Attendance Distribution  
  - School Type Distribution

Interactive Filters
Users can dynamically filter the dashboard by:
- School Type
- Parental Education Level
- Distance from Home
- Gender

Key Insights

- Study hours have a strong positive relationship with exam performance. Students who dedicate more time to studying tend to achieve higher scores.
- Parental involvement positively influences student outcomes. Students with high parental support generally perform better academically.
- Attendance contributes to better academic performance. Consistent attendance supports improved exam results.
- Most students fall within the mid-range score category (60–80). This indicates moderate overall performance across the dataset.
- Motivation level appears to differentiate high-performing students from average performers.
